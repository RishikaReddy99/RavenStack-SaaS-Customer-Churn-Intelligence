# 🗄️ SQL Business Analysis

This folder contains the SQL queries used to investigate customer churn patterns, identify business risks, and support executive decision-making.

## Business Questions Answered

* Which industries experience the highest churn?
* Which acquisition channels retain customers best?
* Which subscription plans drive the greatest revenue loss?
* When are customers most likely to churn?
* What are the primary reasons behind refunds?
