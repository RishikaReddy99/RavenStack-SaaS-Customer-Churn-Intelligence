# Jira Project Management

This folder contains the Agile project management artifacts for the RavenStack Customer Churn Analytics Business Analysis project.

## Tool Used
- Jira Software (Scrum)

## Project Structure

### Epic 1
Business Understanding & Requirements

- Define Project Overview
- Conduct Requirement Elicitation
- Prepare Analytics BRD
- Prepare Analytics FRD

### Epic 2
Data Preparation & Analysis

- Import & Understand Dataset
- Data Validation & Quality Assessment
- SQL Business Analysis
- Excel Business Analysis

### Epic 3
Dashboard Development & Reporting

- Design KPI Dashboard
- Develop Power BI Dashboard
- Dashboard Validation
- Prepare Executive Dashboard Report

### Epic 4
Project Validation & Closure

- User Acceptance Testing (UAT)
- Business Recommendations
- Project Sign-off

## Agile Process

- Scrum Framework
- Sprint Planning
- Sprint Execution
- Sprint Reviews
- Sprint Completion
